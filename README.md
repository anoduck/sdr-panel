# Introduction

This project is part of a larger project called `sdr-scanner` available [here](https://github.com/shajen/rtl-sdr-scanner-cpp). Please familiar with it before starting work.

# Docker

Build

```
git clone https://github.com/shajen/monitor
cd monitor
docker build -t shajen/monitor -f Dockerfile .
```

Run

```
docker run --rm -it shajen/monitor
```

# Contributing

In general don't be afraid to send pull request. Use the "fork-and-pull" Git workflow.

1. **Fork** the repo
2. **Clone** the project to your own machine
3. **Commit** changes to your own branch
4. **Push** your work back up to your fork
5. Submit a **Pull request** so that we can review your changes

NOTE: Be sure to merge the **latest** from **upstream** before making a pull request!
